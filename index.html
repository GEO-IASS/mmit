M.M.I.T

Matlab Medical Imaging Toolset

View project onGitHub

Details

A set of functions developed over the years to help in coding medical-related functions in MATLAB. This code currently requires Jimmy Shen's matlab nifti toolbox, included in /ext, which is under a BSD license.
